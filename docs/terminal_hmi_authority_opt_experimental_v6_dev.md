# Terminal HMI Authority — opt-experimental-v6-dev

In snapshot `opt-experimental-v6-dev`, the terminal launcher is the authoritative HMI surface.

Truth-reflection work for requested/final mode, fallback usage/reason, trust class, and augmentation policy in this snapshot was implemented and validated on the terminal surface.

Any external GUI or web panel outside this snapshot root was out of scope for this pass and is not implied to be in parity.
