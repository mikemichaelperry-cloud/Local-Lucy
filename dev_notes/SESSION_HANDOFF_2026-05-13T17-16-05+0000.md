## What Was Done in This Session

### 1. Schema Migrations — Safe Hardening Pass
**Files:** `tools/router_py/schema_migrations.py`, `tools/router_py/state_manager.py`

- Implemented versioned SQLite schema migration system using `PRAGMA user_version`
- **v1:** Initial schema (namespaces, routes, outcomes, sessions, telemetry, locks + indexes)
- **v2:** Fix sessions table from `session_key` UNIQUE to composite `UNIQUE(namespace_id, session_key)`
- Integrated into `StateManager.__init__()` — migrations run automatically on DB connect
- Removed ad-hoc `_migrate_sessions_unique_constraint()` from `state_manager.py`
- Properties: idempotent, non-destructive, rollback-safe, forward-guarded (newer DB versions raise `RuntimeError`)

**Tests:** `tools/router_py/test_schema_migrations.py` — 6 tests

### 2. Auto-Feedback Threshold Tuning
**Files:** `models/router/auto_feedback.py`, `models/router/background_learner.py`

- **Confidence capping:** Added `_MAX_AUTO_FEEDBACK_CONFIDENCE = 0.5` (env: `LUCY_AUTO_FEEDBACK_MAX_CONFIDENCE`). All auto-feedback entries are capped at this value before logging.
- **Separate thresholds:** `maybe_auto_learn()` now uses distinct thresholds:
  - User feedback: `LUCY_AUTO_LEARN_THRESHOLD` (default **3**)
  - Auto feedback: `LUCY_AUTO_FEEDBACK_THRESHOLD` (default **5**)
- **Trust tiers:** User feedback = high trust (confidence 1.0), auto-feedback = low trust (capped at 0.5)
- **Backward compatibility:** `min_entries` parameter still works as override for user threshold

**Tests:** `models/router/test_auto_feedback.py` — 8 tests

### 3. Voice PTT Controller
**File:** `tools/router_py/voice_runtime.py`

- Added `PTTController` class — async state machine supporting `hold` and `tap` modes
- **Hold mode:** Press starts recording, release stops recording
- **Tap mode:** First tap starts, second tap stops
- Added public API: `ptt_press()`, `ptt_release()`, `is_recording()`
- `_record_audio()` now uses `duration=None` + PTT stop signal instead of fixed 8s
- Timeout guard: `LUCY_VOICE_MAX_SECONDS` hard cap regardless of mode
- Signal handlers properly signal PTT stop on Ctrl+C/SIGTERM

**Tests:** `tools/router_py/test_voice_ptt_controller.py` — 15 tests
