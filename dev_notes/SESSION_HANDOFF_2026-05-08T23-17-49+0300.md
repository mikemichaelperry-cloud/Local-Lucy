## Key Design Decisions

1. **Correct answer > locality.** If LOCAL admits ignorance on a factual query, flag it as a misroute. If AUGMENTED fails, flag fallback to LOCAL.
2. **Heuristics only.** No LLM-based evaluation (too slow, too expensive). Keyword patterns only.
3. **Fail-silent.** Auto-feedback never throws — all calls wrapped in `try/except pass`.
4. **Confidence threshold.** Only ≥ 0.6 confidence entries enter the index.
