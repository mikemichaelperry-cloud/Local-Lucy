# GitHub Push Policy - Local Lucy v10

## ⚠️ WARNING TO AI ASSISTANTS ⚠️

**DO NOT push to GitHub without explicit user confirmation.**

## Protected Content (NEVER PUSH)

The following must remain LOCAL ONLY and must never be pushed to any GitHub repository:

- **Voice Models**: All `.bin`, `.onnx`, `.safetensors` files
  - `ggml-small.en.bin` (Whisper STT model - 465MB)
  - `en_GB-cori-high.onnx` (Piper TTS voice - 108MB)
  - Kokoro TTS model files (~312MB)
  
- **Archive Directories**:
  - `archive_snapshots/` (all retired versions)
  - `archive/versions/` (legacy versions)
  - `quarantine/` (cleanup staging)

- **Pre-v7 Snapshots**:
  - `snapshots/opt-experimental-v1-*`
  - `snapshots/opt-experimental-v2-*`
  - `snapshots/opt-experimental-v3-*`
  - `snapshots/opt-experimental-v4-*`
  - `snapshots/opt-experimental-v5-*`
  - `snapshots/opt-experimental-v6-*`

- **Cache Files**:
  - `cache/` directories
  - `__pycache__/` directories
  - `*.pyc` files
  - HuggingFace cache blobs

- **Large Data Files**: Any file >10MB

## Allowed Push Scenarios

✅ Source code changes (<1MB total)  
✅ Documentation updates (README, AGENTS.md, etc.)  
✅ Configuration files (lucy.conf, etc.)  
✅ Test files (<100KB each)  
✅ VU meter implementation code

## Before Pushing Checklist

1. Verify total push size: `git diff --stat` or `git status`
2. Confirm no model files included: `git status | grep -E "\.(bin|onnx|safetensors)$"`
3. Verify no cache files: `git status | grep -E "(cache|__pycache__)"`
4. Get explicit user confirmation

## When In Doubt

**ASK THE USER BEFORE PUSHING.**

Do not assume. Do not guess. Ask for explicit permission.
