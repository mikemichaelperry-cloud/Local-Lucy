## 1. Issues Fixed

### Voice Reading HTML Tags
**Problem:** Voice was speaking raw HTML like "span style equals color" and "/p/pb2"

**Root Cause:** `sanitize_tts_text()` in `runtime_voice.py` didn't strip HTML tags

**Fix:** Added comprehensive HTML stripping:
- Remove `<script>` and `<style>` elements
- Replace `<br>`, `<p>` with newlines
- Extract link text from `<a>` tags
- Remove all remaining HTML tags
- Decode HTML entities

### Voice Reading Too Much Content
**Problem:** Voice was reading full article descriptions (300+ chars each)

**Fix:** Added voice-optimized news format:
- Voice: "Headline, from Source. Headline, from Source."
- No timestamps, no descriptions, no "Latest World News" headers
- Display format unchanged (full HTML)

### Regional News Not Prioritized
**Problem:** "Israeli news" and "Australian news" queries returned mostly world sources

**Fix:** Implemented feed prioritization:
- Regional sources: 5 articles per feed (was 3)
- World sources: 1 article per feed when region detected (was 3)
