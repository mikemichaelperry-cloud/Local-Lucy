- Mutable metadata → `examples_metadata.json` (untracked, gitignored)

### Stream 2: Unify `request_id` ✅
- `main.py` passes `request_id` into pipeline context
- `PipelineContext` merges it via `extras` (existing frozen dataclass contract)
- `StateWriter` propagates it to JSON payload + SQLite metadata
